## Why This Matters

### 1. **MCP = Any AI Agent Can Use Resupply**

The MCP (Model Context Protocol) server exposes Resupply operations as standardized tools:

- ✅ Works with **any MCP-compatible AI** (Claude Desktop, OpenClaw, custom agents)
- ✅ Agents can **query markets**, **simulate strategies**, **monitor positions**
- ✅ Lowers barrier to entry for **autonomous yield farming**

### 2. **CLI = Power Users + Automation**

Command-line interface for developers and traders:

- ✅ Query markets and APYs (`resupply markets`)
- ✅ Simulate strategies before execution (`resupply simulate`)
- ✅ Check positions and health (`resupply position`)
- ✅ **Scriptable** — integrate into bots, cron jobs, dashboards

### 3. **Brings New Users to Resupply**

- AI agents doing yield farming → more TVL
- Developers building on Resupply → ecosystem growth
- Open source → community contributions
